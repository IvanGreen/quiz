Данное приложение имеет функционал:

Имеется авторизация в системе (регистрации нет). Для шифрования паролей используется BCrypt.

1) Для Администратора:
    1.1) Создавать вопросы 3 типов:
        а) На которые требуется письменный ответ;
        б) На которые требуется выбрать один из нескольких ответов;
        в) На которые требуется выбрать несколько ответов;
    1.2) После создания вопросов их можно добавить в Опрос (Quiz). Можно сделать Quiz из одного или нескольких вопросов.
    1.3) После создания Quiz он находится в сохраненном, но не опубликованном состоянии. Внимание! После публикации
    Quiz удалить не получится. Для публикации требуется зайти под логин и паролем администратора, перейти в вкладку с
    всеми Quiz и отправить Quiz на публикацию.
    1.4) Администратор не может отвечать на Quiz.

2) Для пользователя:
    2.1) Любой, даже не зарегестрированный пользователь может проходить опубликованные Quiz.
    2.2) Только индефицированные пользователи могут посмотреть свои ответы.

Инструкция для запуска приложения:

1) Сохраните приложение к себе на компьютер, в удобное место.
2) Создайте в своей базе данных схему.
3) Требуется отредактировать файл: application.properties , укажите какую базу данных и схему будете использовать,
путь к базе данных, логин и пароль.
4) Требуется отредактировать файл: flyway.conf , внесите, пожалуйста, туда такие же данные как и в прошлом пункте.
5) Запустите flyway (migrate), если Вы правильно внесли все данные в прошлых пунктах, у Вас развернется
база данных со всеми требуемыми данными для старта.
6) Запустите QuizApplication.java
7) Если Вы не изменили базовый адрес, то откройте бразуер и перейдите по ссылке: http://localhost:8189/app/
8) Можно пользоваться приложением!
